Hash Tables

Data structure for fast key-value pair lookup using hash function

What is a Hash Table?

A hash table is a data structure that implements an associative array - a structure that maps keys to values using a hash function. It provides very fast average lookup time.

Key Concepts

  • Hash Function: Converts key to array index
  • Collision: When two keys hash to same index
  • Load Factor: Ratio of entries to array size
  • Chaining: Store collisions in linked list
  • Open Addressing: Find next empty slot for collision

Python Dictionary (Built-in Hash Table)

# Python dictionaries are hash tables
my_dict = {}

# Insert - O(1) average
my_dict['name'] = 'Alice'
my_dict['age'] = 25
my_dict['city'] = 'NYC'

# Access - O(1) average
print(my_dict['name'])  # Output: Alice

# Check if key exists - O(1)
if 'age' in my_dict:
    print(my_dict['age'])  # Output: 25

# Get with default - O(1)
print(my_dict.get('country', 'USA'))  # Output: USA

# Delete - O(1)
del my_dict['city']

# Iterate
for key, value in my_dict.items():
    print(f"{key}: {value}")

# Get all keys - O(1)
keys = my_dict.keys()

# Get all values - O(1)
values = my_dict.values()

Custom Hash Table Implementation

class HashTable:
    def __init__(self, size=10):
        self.size = size
        self.table = [[] for _ in range(size)]
    
    def hash_function(self, key):
        """Simple hash function"""
        return hash(key) % self.size
    
    def insert(self, key, value):
        """Insert key-value pair - O(1) average"""
        hash_val = self.hash_function(key)
        
        # Handle collision using chaining
        for i, (k, v) in enumerate(self.table[hash_val]):
            if k == key:
                self.table[hash_val][i] = (key, value)
                return
        
        self.table[hash_val].append((key, value))
    
    def search(self, key):
        """Search for value by key - O(1) average"""
        hash_val = self.hash_function(key)
        
        for k, v in self.table[hash_val]:
            if k == key:
                return v
        return None
    
    def delete(self, key):
        """Delete key-value pair - O(1) average"""
        hash_val = self.hash_function(key)
        
        for i, (k, v) in enumerate(self.table[hash_val]):
            if k == key:
                del self.table[hash_val][i]
                return True
        return False
    
    def display(self):
        for i, bucket in enumerate(self.table):
            if bucket:
                print(f"Index {i}: {bucket}")

Complexity Analysis

Time Complexity:

Insert: O(1) average, O(n) worst case

Search: O(1) average, O(n) worst case

Delete: O(1) average, O(n) worst case

Space Complexity:

O(n) - Where n is number of key-value pairs

Common Problems

Find Duplicate in Array

def find_duplicate(arr):
    """Find first duplicate using hash table"""
    seen = set()
    for num in arr:
        if num in seen:
            return num
        seen.add(num)
    return None

Two Sum Problem

def two_sum(arr, target):
    """Find two numbers that sum to target"""
    seen = {}
    
    for num in arr:
        complement = target - num
        if complement in seen:
            return [seen[complement], num]
        seen[num] = num
    
    return None

Group Anagrams

def group_anagrams(words):
    """Group anagrams together"""
    anagram_map = {}
    
    for word in words:
        sorted_word = ''.join(sorted(word))
        
        if sorted_word not in anagram_map:
            anagram_map[sorted_word] = []
        
        anagram_map[sorted_word].append(word)
    
    return list(anagram_map.values())
